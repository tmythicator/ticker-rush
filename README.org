* Overview
TickerRush is a real-time stock ladder simulator.
Players start with equal virtual budgets and compete by trading stocks and ETFs.
The goal: grow your portfolio faster than others and climb the leaderboard.

* Status
Early-stage prototype. Frontend base and minimal FastAPI skeleton are being developed.
