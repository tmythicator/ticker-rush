* Ticker Rush — Web Client

A professional trading terminal interface for the *Ticker Rush* project. Built as a high-performance Single Page Application (SPA) using the modern React ecosystem.

** Tech Stack

- *Core:* React 19, Vite, TypeScript
- *Styling:* Tailwind CSS
- *State Management:* TanStack Query (Server state)
- *Data Fetching:* TanStack Query
- *Charts:* [[https://github.com/tradingview/lightweight-charts][Lightweight Charts v5]]
- *Icons:* [[https://lucide.dev][Lucide React]]

**  Local Setup

Ensure you have *Node.js 18+* installed.

1. *Install dependencies:*
   #+BEGIN_SRC bash
   npm install
   #+END_SRC

2. *Start the development server:*
   #+BEGIN_SRC bash
   npm run dev
   #+END_SRC

3. *Open the application:*
   Navigate to [[http://localhost:5173]] in your browser.

**  Project Structure

- ~src/pages~ — Main views (Dashboard, Leaderboard, Profile)
- ~src/components~ — Atomic UI components
- ~src/lib~ — API clients and utilities
